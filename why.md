  <!-- The Solution -->
  <section class="about-section">
    <h2 class="about-section-title">🤖 The Solution: Copilot Cloud Agent</h2>
    <p class="about-section-desc">GitHub Copilot Cloud Agent is an <strong>autonomous AI agent</strong> that runs directly in GitHub. You describe what you need in plain language — the agent does the rest.</p>
    <ol class="step-list">
      <li>
        <strong>Describe your request in a GitHub Issue</strong><br>
        Open a structured form and tell the agent what you want: "Summarize this week's Azure Blog posts" or "Compare GitHub Copilot updates from the last 7 days." Select a specialist persona, tone, and language.
      </li>
      <li>
        <strong>Assign to <code>@copilot</code></strong><br>
        The Cloud Agent activates autonomously. No API keys, no pipeline setup, no infrastructure to manage.
      </li>
      <li>
        <strong>Agent fetches real-time content</strong><br>
        Copilot visits the actual websites — Microsoft blogs, GitHub Changelog, security feeds — and reads the live content. This is <strong>not training data</strong>; it's real-time information retrieval.
      </li>
      <li>
        <strong>Agent writes a professional report</strong><br>
        The report is formatted with executive summaries, key takeaways, source links, and bilingual support. It's submitted as a Pull Request for your review.
      </li>
      <li>
        <strong>Merge → Published</strong><br>
        One click to merge. The report appears on your team's dashboard instantly — searchable, shareable, and always up to date.
      </li>
    </ol>
  </section>

    <h2 class="about-section-title">🇯🇵 なぜ Copilot Cloud Agent なのか？</h2>
    <p><strong>GitHub Copilot Cloud Agent</strong> は、GitHub上で自律的に動作するAIエージェントです。GitHub Issueに調査したい内容を記述し、<code>@copilot</code>に割り当てるだけで、エージェントがリアルタイムに情報を取得・分析し、プロフェッショナルなレポートを作成します。</p>

    <h3 class="about-subsection-title">エンタープライズにおける価値</h3>
    <ul style="line-height: 2; font-size: 15px;">
      <li><strong>時間の削減</strong> — 毎週3〜4時間の手動リサーチが、5分のIssue作成に置き換わります</li>
      <li><strong>情報の見落とし防止</strong> — Microsoft・GitHub の13以上のドメインからリアルタイムに情報を収集</li>
      <li><strong>一貫したフォーマット</strong> — 10種類の専門ペルソナと7種類の文体で、目的に合ったレポートを自動生成</li>
      <li><strong>バイリンガル対応</strong> — 英語・日本語の両方に対応し、グローバルチームでの活用が可能</li>
      <li><strong>コーディング不要</strong> — セットアップもメンテナンスも不要。GitHub Issueだけで完結します</li>
    </ul>
