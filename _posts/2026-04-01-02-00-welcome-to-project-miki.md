## Executive Summary

Welcome to **Project Miki**, an intelligence dashboard curated by GitHub Copilot Coding Agent. This sample report demonstrates the format that future reports will follow.

## How This Dashboard Works

When you want to research something — whether it's the latest Microsoft Blog posts, GitHub Copilot updates, or Azure security announcements — you simply:

1. **Create a GitHub Issue** describing what you want to know
2. **Assign it to `@copilot`** — the Coding Agent picks it up
3. **Wait for the PR** — Copilot fetches real content, summarizes it, and writes a report
4. **Merge** — the report appears here on the dashboard

## Available Information Sources

The Coding Agent can fetch fresh content from **13 Microsoft and GitHub domains**, including:

- **Microsoft:** Official Blog, AI Blog, Dev Blogs, Azure Blog, Security Blog, News Center Japan, and more
- **GitHub:** GitHub Blog, Changelog, Docs, Universe

## Example Issue Descriptions

Here are some examples of what you can ask:

- "Summarize this week's posts from the Official Microsoft Blog"
- "What's new in GitHub Copilot? Check the GitHub Blog and Changelog"
- "Microsoft AI Blog の最新記事をまとめてください"
- "Compare Azure Blog and Security Blog articles about Zero Trust"

## Key Takeaways

- This dashboard is **fully automated** — no code to install, no APIs to configure
- Reports are **permanently saved** as markdown files in this repository
- Anyone with a **GitHub Copilot subscription** can create new reports
